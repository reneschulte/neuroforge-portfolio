# Claw NeuroForge — OpenClaw Portfolio Site

A single-page marketing site showcasing Claw NeuroForge, a boutique studio delivering OpenClaw automation, agent engineering, and ops enablement services.

## Tech stack
- Vanilla HTML, CSS (Space Grotesk), and a sprinkle of JS for UX polish
- Responsive, dark-mode native layout with semantic sections
- Deployed automatically with GitHub Pages via the included workflow

## Local development
```bash
cd neuroforge-portfolio
python3 -m http.server 8000
# visit http://localhost:8000
```

## Deployment
Push to `main` — the GitHub Actions workflow publishes to `gh-pages`, powering GitHub Pages at `https://<username>.github.io/neuroforge-portfolio/`.
